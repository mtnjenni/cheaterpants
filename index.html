HTML & CSS 101

HTML Structure

Tag Description
<!DOCTYPEhtml> First line of code declaring document type to the browser and which web standards to apply.
<html> Document type that encloses all the code and content that makes up the webpage.
<head> Contains non-displayed general info about the page and meta data like the Title and links to stylesheets.
<title> In the head of the document, defines a title that is displayed in the tab or title bar.
<body> Page content of semantic html document. Only one per document.
<header> Introductory page section that can include navigation, logo, heading tags, search bar, etc.
<a> Anchor tag used to link internal and external pages or content on the same page.
<main> Primary content within the body of a webpage.
<section> Defines an area of the page with the same theme such as chapters or headings.
<aside> Page section of content indirectly related to the main page such as related links or advertisements etc.
<footer> Page section at the end of the content block with infomation like copyright or links to relevant pages.

HTML Content

Tag Description
<nav> Defines a block of navigation like menus or or table of contents.
<select> Creates a drop down list for navigation.
<h1>-<h6> Leveled heading elements with h1 the highest level and h6 the lowest level for content text.
<article> Self-contained part of page that could be published elsewhere like blogs or online articles.
<p> Paragraph displays a block of text.
<br> Line break. Only requires opening tag.
<li> Element containg content of a list.
<ol> Ordered sequential list. Numbered by default. Encapsulates list content.
<ul> Unordered list. Bulleted by default. Encapsulates list content.
<figure> Encapsulates media such as an image, diagram, or code sinppet.
<figcaption> Describes the media within the figure.
<img> Embeds images in documents with no closing tag.
<video> Embeds media player for playback.
<embed> Embeds external content such as video, audio, images, etc.
<link> Connects current page with an external page.
<table> Content to represent a 2D table made of rows and columns.
<tr> Adds a row to the table before adding data and headings.
<thead> Defines headings of table columns encapsulated in table rows.
<th> Adds titles to rows and columns, enclosed in <tr> table row element within the table head.
<tbody> Contains all table data
<td> Table data nested in a table row <tr> to add a cell of data to a table.
<tfoot> Uses table row to give footer content or to sumerize content at the end of a table.
<div> Container to divide into sections of non-semantic html.
<span> Inline container of text for styling. non-semantic html.
<em> Emphasizes text, italisied by default. non-semantic html.
<strong> Highlights important, serious, urgent text, bold by default. non-semantic html.

CSS Box-Model

Property: Value; Description
box-sizing: border-box; Height and width of element. Includes: content box, padding, and borders. Resets the default (content-box)
margin: Defines the space between the border and the outside edge of the box.
border: width style color; Defines the border around the content and padding. Properties defined in same line of code separated by a space.
border-radius: %; Modifies corners to create rounded boxes. 50% = circle.
padding: Defines the space between the content and the border.
width: Defines the width of the content, unless box-sizing is set to border-box.
height: Defines the height of the content, unless box-sizing is set to border-box.
min-width/height: Sets the minimum width/height of the element.
max-width/height: Sets the maximum width/height of the element.
overflow: hidden; Hides any content that spills outside the box.
overflow: scroll; A scroll bar is added to view content that spills outside the box.
overflow: visible; Content will be seen outside the box. Default.

CSS Positioning & Display

Property: Value; Description
position: static; Default, element remains in flow of document and appears in markup location.
position: relative; Requires offset properties to move element away from default position. Creates reference positoin for absolute child elements.
position: absolute; Removes element from flow of document and places in relation to closest relative positioned ancestor element.
position: fixed; Removes element from flow of document and is always relative to the document markup location.
position: sticky; The element is treated as relative until a specific point or threshold is met where the element then becomes fixed.
z-index: Controls how far back or forward elements appear (depth).
display: block; Fills width of page creating a line break after element and can be set to height necessary to accomodate content.
display: inline; Takes up as little space as possible. Occupy height and width of content and does not move. Default.
display: inline-block; Content can be placed next to each other and can specify height and width.
clear: Controls how elements behave when they bump into each other.
float: In a static or relative position, moves an element left or right of its container. Used to wrap text around an image.
